## 1. Install

To install and save the latest version of Svelvet in your package.json dependencies, run the command below using your preferred package directory.

<CodeGroup>

```sh npm
npm install svelvet
```

```sh yarn
yarn add svelvet
```

```sh pnpm
pnpm add svelvet
```

</CodeGroup>
